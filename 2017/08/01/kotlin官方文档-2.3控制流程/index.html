if 表达式

if是一个表达式，即它返回一个值，所以Kotlin没有三元运算符（condition？then：else），有if就够了。

如果将if用作表达式而不是语句（例如，返回值或将其分配给变量），则表达式需要具有else分支。

When 表达式

when会连续匹配所有分支的参数，直到满足某些分支条件为止。可以用做表达式或声明。如果将其用作表达式，则满足分支的值成为整个表达式的值。如果用作声明，则忽略单个分支的值。（与 if 表达式一样，每个分支是一个语句块，而且它的值就是最后一个表达式的值)

如果其他分支条件都不满足, 则默认匹配 else 分支。如果把 when 做为表达式， else 分支是强制的，除非编译器可以证明分支条件已经覆盖所有可能性。除非编译器可以证明所有可能的情况都覆盖了分支条件。
